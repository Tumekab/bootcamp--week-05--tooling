# Install `node`

## 1) First install Node

### Mac

```bash
brew install node
```

### Windows

```bash
curl -sL "https://deb.nodesource.com/setup_14.x" | sudo -E bash -
sudo apt-get install -y nodejs
```

## 2) Now check Node installed

Run `node -v`

We expect something like:

```bash
v14.7.1
```
